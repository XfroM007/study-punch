---

## 一 · 目标

高三上学期古文背诵 + 英语单词打卡工具，**面向外人使用**（非老刘自用），PC/手机通用，离线运行。

**外人版核心要求**（2026-07-15 老刘拍板）：
- **零门槛**：双击 `index.html` 即可使用，无任何环境依赖
- **离线完整可用**：图标/字体/数据全部内嵌，不依赖 CDN/外网
- **专业感**：内容来源权威、UI 精致、交互流畅
- **品牌化**：应用名、启动页、关于页要像一个正式产品
- **可分享**：可导出/导入数据备份；老刘可分发单个 HTML 文件给别人

## 二 · 内容范围（已锁）

| 维度 | 数量 | 来源 |
|---|---|---|
| 古文篇目 | 14-16 篇 | 人教/部编版高三上必背篇目（**内容 agent 需先列清单给我过目**，未经审核不入库） |
| 英语单词 | 600-800 词 | 高考考纲核心词汇（高三上学期重点 800 词）（**同上**） |
| 每日打卡量 | 1 篇古文 + 20 词 | 老刘拍板 |

## 三 · 打卡机制

- 每日 1 篇古文 + 20 词，不可跳、不可漏
- 古文：逐句核对（输入/选择 + 错字高亮）
- 单词：英→中 / 中→英 双向测验
- 进度：连续天数 streak / 累计天数 / 完读篇目数 / 累计单词数
- 成就卡牌收集：达成里程碑解锁卡牌（如"完读 1/3/5/全"）
- 积分：每日打卡 +10，连续 7 天 +20，连续 30 天 +50

## 四 · UI 规范 · 深空玻璃拟态

| 元素 | 规范 |
|---|---|
| 主色 | 深空蓝黑 `#0a0e27` → 紫蓝 `#1e1b4b` 渐变 |
| 玻璃面板 | `backdrop-filter: blur(20px)` + 半透明白 `rgba(255,255,255,0.06)` + 1px 边框 `rgba(255,255,255,0.12)` |
| 强调色 | 霓虹青 `#00f5ff` + 紫粉 `#ff006e`（连胜/进度用霓虹青，完读/成就用紫粉）|
| 字体 | 思源黑体（系统兜底 -apple-system / "PingFang SC"） |
| 圆角 | 16px / 24px 两档 |
| 阴影 | 多层：`0 8px 32px rgba(0,0,0,0.4)` + 内发光 `inset 0 1px 0 rgba(255,255,255,0.1)` |
| 动效 | 平滑过渡 300ms cubic-bezier(0.4, 0, 0.2, 1)；打卡成功用粒子扩散 |

## 五 · 数据持久化

- 全部用 `localStorage`，key 命名 `study-punch:v1:*`
- 首次进入初始化空数据集 + 默认设置
- 提供"导出 JSON" / "导入 JSON" / "重置"三按钮（设置面板）

## 九 · 风险与边界

| 风险 | 对策 |
|---|---|
| 内容准确性（古文原文/英文词义） | 主 agent 逐条审核，不放过一字 |
| 古文标点/异体字差异 | 默认采用权威版本（人教社课本） |
| 移动端适配 | CSS 用 rem + 媒体查询，关键交互 44px 触控 |
| 离线图标 | 用 inline SVG 或 emoji，不依赖 CDN |
